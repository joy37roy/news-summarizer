# AI News Summarizer 📰

My third AI project — summarizes any text and extracts key points!

## Features
- Paste any text and get a 3 sentence summary
- Extracts 5 key points automatically
- Identifies the main topic
- Saves all summaries to JSON file
- Loop — summarize multiple texts in one session
- Type 'quit' to exit cleanly

## Technologies Used
- Python
- Groq API (llama-3.3-70b-versatile)

## What I Learned Building This
- Working with longer text input
- Structured AI responses
- JSON data storage
- Error handling with try/except

## How to Run
1. Install groq: pip install groq
2. Add your Groq API key
3. Run: python summerizer.py
